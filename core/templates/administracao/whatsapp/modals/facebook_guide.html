Guia: Como Obter Credenciais do Facebook
Importante: Você precisa ter uma conta Facebook Business e um aplicativo configurado para WhatsApp Business API.

  1. Acesse: https://developers.facebook.com/
  2. Faça login com sua conta Facebook Business
  3. Clique em "Meus Apps" no canto superior direito
  4. Selecione seu app ou clique em "Criar App"

  1. No painel do seu app, clique em "WhatsApp" no menu lateral
  2. Vá para "API Setup"
  3. Aqui você encontrará todas as credenciais necessárias
  4. Configure seu número de telefone
  5. Gere o Access Token permanente

Onde Encontrar Campo no Sistema Descrição
App Dashboard → ID do App ID do Aplicativo Número longo visível no topo da página
App Dashboard → Configurações Básicas Chave Secreta do Aplicativo Clique em "Mostrar" ao lado de "Chave Secreta do App"
WhatsApp → API Setup Phone Number ID ID do número de telefone configurado
WhatsApp → API Setup Business Account ID ID da conta de negócios
WhatsApp → API Setup Token de Acesso Gere um token permanente (não temporário)

O webhook é essencial para receber mensagens!
  1. Vá para WhatsApp → Configuration
  2. Em "Webhook", clique em "Edit"
  3. Configure:
    • Callback URL: https://seudominio.com/webhook/whatsapp/ID_DA_CONTA/
    • Verify Token: Um token personalizado (mesmo do campo "Token de Verificação Webhook")
  4. Marque os eventos: messages, message_deliveries, message_reads
  5. Clique em "Verify and Save"
Dica Importantássima!

Mantenha suas credenciais seguras e nunca as compartilhe. Use apenas em ambiente seguro (HTTPS).

O Access Token deve ser permanente, não o token temporário de 24 horas.
