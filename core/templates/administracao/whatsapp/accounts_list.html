Como obter credenciais da API WhatsApp Business
Você precisará de uma conta Meta for Developers e um aplicativo configurado para usar a API do WhatsApp Business.

  1. Acesse developers.facebook.com
  2. Clique em "Começar" ou "Get Started"
  3. Faça login com sua conta Facebook/Meta
  4. Complete o registro como desenvolvedor

  1. No painel do desenvolvedor, clique em "Criar App"
  2. Escolha "Business" como tipo de aplicativo
  3. Preencha os dados do aplicativo
  4. Após criar, anote o App ID e App Secret
O App Secret é sensível - mantenha-o seguro!

  1. No seu app, vá para "Adicionar Produto"
  2. Encontre e adicione "WhatsApp"
  3. Clique em "Configurar" no WhatsApp
  4. Crie ou vincule uma conta do WhatsApp Business
  5. Anote o Business Account ID

  1. Na configuração do WhatsApp, vá para "API Setup"
  2. Você verá uma lista de números de telefone
  3. Copie o Phone Number ID do número que deseja usar
  4. Este ID é diferente do número de telefone real

  1. No "API Setup", clique em "Generate Access Token"
  2. Copie o token gerado
  3. Para produção, você precisará gerar um token permanente
Importante: Tokens temporários expiram em 24 horas. Para uso em produção, configure um token permanente.

  1. Defina um Webhook Verify Token (pode ser qualquer string segura)
  2. Use esse token no campo correspondente do formulário
  3. Após salvar a conta, copie a URL do webhook gerada
  4. No Facebook, configure essa URL e o verify token
A URL do webhook será exibida após criar a conta.
